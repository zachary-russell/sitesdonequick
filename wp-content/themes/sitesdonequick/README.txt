GENESISTHE.ME DEVELOPER STARTER

=============  NOTE: =============
YOU MUST COMPILE A STYLE.CSS FILE FROM THE SASS IF YOU WANT TO MAINTAIN THE INTEGRITY OF THE SASS FILES. ONLY MAKE STYLE CHANGES TO THE SASS FILES.
THIS IS A DEVELOPER TOOL.


INSTALL
1. Add New Theme with the .zip file. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the theme.
4. You're a developer, so have fun!
